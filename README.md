# H5 → SEG-Y 转换与绘图

这是一个用于 FEBUS H5 数据转换为 SEG-Y 并快速绘图的 GUI 工具。

## 依赖
- Python 3.10+
- `numpy`, `h5py`, `segyio`, `matplotlib`, `tkinter`

如果你使用 `uv`：
```bash
uv sync
```

## 启动 GUI
```bash
python h5_to_segy_gui.py
```

## 操作步骤（推荐顺序）
1. 点击 **“打开文件夹（扫描 H5）”**，选择包含 `.h5/.hdf5` 的目录。
2. 在左侧列表选择一个 H5 文件。
3. 点击 **“转换为 SEG-Y”** 生成单个 `.segy` 文件。
4. 点击 **“绘制 SEG-Y”** 预览当前文件的转换结果。

## 批量转换
- 点击 **“转换所有文件”**，将当前列表中的所有 H5 批量转换为 SEG-Y。
- 输出目录可通过 **“选择输出目录”** 设置，未选择则默认写到 H5 同目录。

## 输出文件
- 输出文件名：`原始文件名.segy`
- `dt` 自动从 FEBUS H5 内部属性读取，无需手工输入。

## 常见问题
- 如果提示 “只有 3600 字节头，没有 trace”，说明输出 SEG-Y 里没有实际 trace 数据，需要重新转换或检查输入数据。
